Flow of Control and Data

The diagram above explains how a search query for a bodybuilding product is executed. Below are the sequence of Events.

  1. User types in the name of supplement they wants to buy on the store page and select the number of results he wants to diplay.
  2. Store.aspx calls the parser with the provided search query to call the Ebay Search Webservice.
  3. The Parser Returns data i.e. products details, after parsing the response of the webservice.
  4. Store.aspx displays the supplement deatils corresponding to the user search query.
