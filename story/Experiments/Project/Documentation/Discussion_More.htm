Flow of Control and Data

The diagram above explains how the query for displaying the fitness article is executed on the More.aspx page.

  1. User click on More Link from the Nav bar.
  2. More.aspx calls the SOAP Client which call the Deepfitness Article Search Webservice.
  3. The SOAP Client returns the details about the articles, after parsing the response of the webservice.
  4. More.aspx displays the Title,description for the article. A button to read the full article is also displayed on the page more.aspx

Random Fitness articles are displayed every time user refers to the More page or even refreshes the page.
