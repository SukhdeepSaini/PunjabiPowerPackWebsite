Workout.aspx

The workout page displays static data which is hardcoded and the results are shown as per the values selected by the user in the dropdowns present on the workout page. I have used a iframe tag on the page which dynamically changes the value of the source attribute for the iframe based on the selection made by the user for the excercise video he wants to see.

Output Screen

DATA

Static data displayed using C# and JQuery

Further Action

User is provided with the workout schedules and the excercise videos based on the selection he made while searching the content.

Future Actions

The data on the page will be made dynamic using some API and more customizable options for workouts will be provided to the user.
