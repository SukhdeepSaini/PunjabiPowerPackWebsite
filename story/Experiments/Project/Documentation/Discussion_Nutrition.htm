Nutrition.aspx

Flow of Control and Data

The diagram above explains how a search query for healthy diet recipes is executed. Below are the sequence of Events.

  1. User types in the name of the recipe for which he wants to get the details on Nutrition page.
  2. Nutrition.aspx calls the parser with the provided search query to call the Yummly recipe Webservice.
  3. The Parser Returns data i.e. Recipe details, after parsing the response of the webservice.
  4. Nutrition.aspx displays the Thumbnail and the links to the corresponding recipe on yummly as per the user search query.
