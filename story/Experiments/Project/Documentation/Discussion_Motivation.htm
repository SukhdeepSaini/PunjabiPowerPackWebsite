Flow of Control and Data

The diagram above explains how a search query for a bodybuilding Motivational Videos is executed. Below are the sequence of Events.

  1. User types in the Video name that they wants to search on the Motivation page.
  2. Motivation.aspx calls the parser with the provided search query to call the Youtube Search Webservice.
  3. The Parser Returns data i.e. Video details, after parsing the response of the webservice.
  4. Motivation.aspx displays the Videos tags and the links to the corresponding videos on youtube as per the user search query.
